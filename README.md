# Ghibli App

## Overview

This project is a **Ghibli App** that uses the Studio Ghibli API to retrieve movie data:

https://ghibliapi.vercel.app

The application displays a list of Studio Ghibli movies.  
When a user selects a movie, they are navigated to a detail screen where they can view additional information about the selected film.

---

## Features

- Display a list of Ghibli movies
- Navigate to a movie detail screen
- Retrieve data from a REST API
- Modern Android architecture
- Clean and scalable project structure

---

## Tech Stack

The project is built using:

- MVVM Architecture
- Jetpack Compose
- Jetpack Navigation 2
- Kotlin Coroutines
- Dagger Hilt
- Retrofit
